I recently finished my MS in Data Analytics Engineering at Northeastern (GPA: 3.89) and I've spent the last 2+ years building data and ML systems professionally.

## 🚀 Projects

### [MarketMind — AI-Powered Daily Stock Briefing Platform](https://github.com/ragulnarayanan/marketmind) &nbsp;|&nbsp; [Live Demo](https://ragulnarayanan-marketmind-app-u3eic9.streamlit.app/)
Built a 5-agent LLM pipeline that reads overnight financial news, SEC filings, and market data to generate personalized daily briefs for investors. RAG over SEC filings, real-time financial metrics, and narrated audio briefs. Deployed on GCP Cloud Run with daily scheduling.
- **Stack:** LangChain, OpenAI GPT, Gemini, Qdrant Cloud, Firestore, GCP Cloud Run, OpenAI TTS, Streamlit

### [MacroMarket ELT — Azure-Native Financial Data Pipeline](https://github.com/ragulnarayanan/macromarket-elt) &nbsp;|&nbsp; [Docs](https://ragulnarayanan.github.io/macromarket-elt/)
End-to-end ELT pipeline ingesting stock, macro, and crypto data from 4 sources into Snowflake via medallion architecture. 20+ dbt models, 30+ data quality tests, MCP server for LLM queries, and a 4-page Streamlit dashboard.
- **Stack:** Python, Snowflake, dbt, Azure (ADLS, Data Factory, Key Vault, DevOps), MCP, Streamlit, Plotly

### [Secure Local DB Agent — Offline Text-to-SQL with an AST-Based Safety Layer](https://github.com/ragulnarayanan/secure-local-db-agent)
Fully offline text-to-SQL agent on local LLMs via Ollama with zero cloud egress. SQL guardrail layer using sqlglot AST parsing blocks injection and out-of-schema queries. Benchmarked 3 local LLMs — Llama 3.1 8B achieved 85% accuracy.
- **Stack:** Ollama, Llama 3.1, Qwen2.5-Coder, sqlglot, Pydantic, SQLite, Streamlit, pytest

### [EchoAI — End-to-End MLOps Pipeline for Automated Review Response](https://github.com/abhisek-ai/echo-ai)
Automated data pipelines for scraping, preprocessing, and model training on 100K+ customer reviews. Trains and compares sentiment models, selects the best performer, monitors for bias and drift. FLAN-T5 for response generation with Streamlit UI.
- **Stack:** Apache Airflow, BigQuery, MLflow, DVC, FLAN-T5, GCP Cloud Run, Docker, CI/CD, Streamlit

### [Amazon Electronics Recommendation System](https://github.com/ragulnarayanan/amazon-electronics-recommendation-system)
Hybrid search pipeline over 6.7M+ records with finetuned Pegasus and RoBERTa on AWS SageMaker (4x A10G GPUs). Served via FastAPI at sub-500ms latency.
- **Stack:** PyTorch, FastAPI, Qdrant, Pegasus, RoBERTa, BGE Embeddings, BM25, AWS SageMaker, GCP

### [CNN Classification + YOLOv8 OBB Detection](https://github.com/ragulnarayanan/cnn-object-classification) &nbsp;|&nbsp; [YOLOv8 Repo](https://github.com/ragulnarayanan/yolov8-obb-detection)
Built and evaluated 20 CNN architectures for 39-class image classification, achieving 96.59% accuracy. Annotated datasets using Roboflow and trained YOLOv8 oriented bounding box models for multi-object detection.
- **Stack:** TensorFlow, Keras, YOLOv8, Roboflow, OpenCV, Google Colab GPU

### [Amazon Books Data Pipeline](https://github.com/ragulnarayanan/amazon-books-etl)
Containerized ETL pipeline automating daily scraping, transformation, and storage of Amazon book data. Modular Airflow DAGs with data quality checks and deduplication.
- **Stack:** Apache Airflow, PostgreSQL, Docker Compose, Python, Git

### [Customer Churn Analysis & Prediction](https://github.com/ragulnarayanan/customer-churn-analysis)
EDA on 7,043 telecom customers with classification models (Logistic Regression, SVM, XGBoost) achieving ROC-AUC of 0.812 and actionable churn reduction recommendations.
- **Stack:** Scikit-learn, SMOTE, PCA, Matplotlib, Python

### [Autonomous Perception Stack — Camera-LiDAR Fusion](https://github.com/ragulnarayanan/autonomous-perception-stack)
End-to-end AV perception pipeline on nuScenes — YOLOv8 segmentation, LiDAR projection, camera-LiDAR fusion, 3D localization, DBSCAN clustering, multi-object tracking, trajectory prediction, and BEV mapping.
- **Stack:** YOLOv8, OpenCV, PyTorch, nuScenes SDK, DBSCAN, Matplotlib

### [nuScenes Failure Atlas — Perception Model Error Analysis](https://github.com/ragulnarayanan/nuscenes-failure-atlas)
Analyzed 187K ground-truth objects to find where a 3D detector fails and why. Built a miss-predictor at 0.94 ROC-AUC. 76% of all failures concentrated in one addressable slice.
- **Stack:** Scikit-learn, pandas, numpy, matplotlib, Google Colab

### [PerceptionOps — Production Object Detection with Monitoring](https://github.com/ragulnarayanan/perception-ops)
YOLOv8 detection service with FastAPI, Prometheus metrics, Grafana dashboards, and Docker Compose deployment. Image, video, and live webcam inference with CI/CD via GitHub Actions.
- **Stack:** YOLOv8, FastAPI, Prometheus, Grafana, Docker Compose, Streamlit, Railway
